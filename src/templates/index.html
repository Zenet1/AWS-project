Universidad API

Esta API es utilizada como un CRUD para las entidades 'Alumno' y 'Profesor'. Se pueden recuperar todos los registros o registros particulares, insertar, actualizar y eliminar registros de ambas entidades.

Estos son los siguientes Endpoints disponibles:

Alumnos:

  • GET '/alumnos': Devuelve una lista de todos los alumnos registrados
  • GET '/alumnos/{id}': Devuelve un único registro del alumno con la id especificada
  • POST '/alumnos': Inserta un registro de un alumno con los datos especificados en la petición
  • PUT '/alumnos/{id}': Actualiza el registro de un alumno particular, con los datos especificados en la petición
  • DELETE '/alumnos/{id}': Elimina el registro del alumno con la id especificada

Profesores:

  • GET '/profesores': Devuelve una lista de todos los profesores registrados
  • GET '/profesores/{id}': Devuelve un único registro del profesores con la id especificada
  • POST '/profesores': Inserta un registro de un profesor con los datos especificados en la petición
  • PUT '/profesores/{id}': Actualiza el registro de un profesor particular, con los datos especificados en la petición
  • DELETE '/profesores/{id}': Elimina el registro del profesor con la id especificada
